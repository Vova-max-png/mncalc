**MNCALC** - A small calculator for Minecraft.

# Instalation

## Cargo:
```
cargo install mncalc
```

# Usage

Use the command mncalc:
```
mncalc <width> <height> <count of countries>
```

Examples:
```
mncalc 2000 2000 4
mncalc 8000 8000 6
mncalc 12576 48756 7
```
